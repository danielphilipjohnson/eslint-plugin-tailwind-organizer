# Security Policy

## Supported Versions

The following versions of `eslint-plugin-tailwind-organizer` receive security updates:

| Version | Supported |
|--------|-----------|
| Latest | ✔ Active |
| Previous minor versions | ✔ Critical fixes only |
| Pre-release tags (`canary`, `alpha`, `beta`) | ❌ Not guaranteed |

If you rely on older versions, please upgrade to the latest stable release.

---

## Reporting a Vulnerability

If you discover a security issue **DO NOT** open a public GitHub issue.

Instead, please report it privately:

- Use GitHub’s *“Report a Vulnerability”* feature under the **Security** tab

---

## Scope

This project handles:

- Parsing AST nodes  
- Automatic formatting  
- ESLint integration  

It does *not* process external user input or run untrusted code, but dependency-related risks still apply.

---

## Safe Disclosure

We greatly appreciate responsible disclosure and will credit contributors who help keep the ecosystem secure.
